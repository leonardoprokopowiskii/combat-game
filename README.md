# 🎮 Combat Game (Python)

Um jogo simples de combate em turnos feito em **Python**, utilizando conceitos de **Programação Orientada a Objetos (POO)**.

O jogador controla um herói que enfrenta um inimigo em batalhas baseadas em escolhas, podendo usar ataques normais ou especiais.

---

## 🚀 Funcionalidades

- Sistema de combate em turnos
- Ataque normal com dano aleatório
- Ataque especial mais forte
- Sistema de vida (HP)
- Classes com herança (Hero e Enemy)
- Encapsulamento de atributos
- Interface simples via terminal

---

## 🧠 Conceitos aplicados

Este projeto utiliza:

- ✔️ Classes e Objetos  
- ✔️ Herança  
- ✔️ Encapsulamento  
- ✔️ Polimorfismo (override de métodos)  
- ✔️ Modularização básica  
- ✔️ Uso de `random` para dano dinâmico
